חלק 4: היכרות עם שאילתות

אז בניתן טבלה עם 200 רופאות ורופאים, והמידע כבר שם.
אבל איך שואלים שאלות על הטבלה?
איך אפשר למשל לראות רק את הרופאים שמתמחים ברפואת ילדים? או רק את אלה שצעירים מגיל 40?

כדי לעשות את זה – נכיר כלי נהדר ופשוט לשימוש: פונקציית QUERY של Google Sheets.

מה זה QUERY ?

QUERY היא פונקציה מיוחדת בגיליונות Google שמאפשרת לנו לשלוף נתונים מתוך טבלה – לפי כללים שאנחנו קובעות. במילים אחרות: היא מאפשרת לנו לשאול שאלות על הטבלה ולקבל את התשובות – בלי לעבור על כל הטבלה ידנית.

לדוגמה:

  • ״תציגי רק את הרופאים שעוסקים בנוירולוגיה״
  • ״תציגי רק את מי שמרוויח\ה מעל 350,000 דולר״
  • ״תמצאי את הרופא הכי צעיר שמתמחה ברפואת ילדים״

איך משתמשים בפונקציה?

בוחרים תא ריק בטבלה, וכותבים בו:
=QUERY(data, query, headers)

מה פירוש כל חלק בפונקציה?

  • data – טווח התאים שמכיל את הטבלה שעליה אנחנו רוצות לעבוד (למשל A1:E10)
  • query – השאלה שאנחנו שואלות על הנתונים, בניסוח פשוט באנגלית
  • headers – כמה שורות עליונות בטווח הן שורות כותרת (כלומר: שמות השדות). בדרך כלל: 1

דוגמה מתוך הטבלה שלנו:

נניח שאנחנו רוצות לדעת אילו רופאים מרוויחים מעל 300,000 דולר בשנה.
נשתמש בפונקציה הבאה:

=QUERY(A1:E10, "SELECT A, B WHERE D > 300000", 1)

מה המשמעות של כל חלק?

  • A1:E10 – זה טווח הטבלה שעליו מופעלת השאילתה. במקרה הזה: רק עשר השורות הראשונות בגיליון, כלומר 9 רשומות (שורה 1 היא כותרת, שורות 2–10 מכילות מידע על רופאים).
  • SELECT A, B – אנחנו בוחרות להציג בתוצאה רק את עמודה A (שם פרטי) ועמודה B (שם משפחה).
  • WHERE D > 300000 – אנחנו רוצות לסנן ולהציג רק את מי שמשתכרים מעל 300,000 דולר. D היא העמודה של השכר השנתי (Salary).

מה נקבל כתוצאה?

הפונקציה תציג לנו רק את השמות הפרטיים ושמות המשפחה של הרופאים שמרוויחים יותר מ־300,000 דולר לשנה – מתוך הטווח A1:E10 בלבד. כלומר, היא לא מסתכלת על כל הטבלה – אלא רק על החלק שבחרנו (A1:E10).

ה- query היא SELECT A, B WHERE D > 300000.
הפעולה SELECT A, B אומרת בעצם שבתוצאה נציג את עמודות A ו-B – במקרה שלנו את השם הפרטי ואת שם המשפחה של הרופא.
הפעולה WHERE מסננת את הרשומות ומציגה רק את הרשומות בהן D > 300000 – כלומר רק את הרופאים שמרוויחים יותר מ־300 אלף דולר בשנה.

לסיכום, השאילתה מחזירה את השמות הפרטיים ושמות המשפחה של הרופאים שמרוויחים מעל 300 אלף דולר בשנה.
והפונקציה QUERY מפעילה את השאילתה הזו רק על 18 הרשומות הראשונות בטבלה.
